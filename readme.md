# DevCamper API

> Backend Api for DevCamper application. which is a bootcamp directory website

## Usage

Rename "config/.env.env" to "config/.env" and update the values / settings to your own

## Install Dependencies
```
npm install
```
## Run App
```
# Run in dev mode
npm run dev

#run in prod mode
npm start
```
